# SUSS Fine Dining

A sophisticated, minimal web landing page for a fine dining restaurant reservation system.

## Features
- **Minimalist Aesthetic:** Avoiding overused AI-style gradients for a clean, professional look.
- **Reservation Form:** Simple UI for booking tables with date and time selection.
- **Responsive Design:** Works across mobile and desktop devices.

## Setup
Simply open `index.html` in your browser to view the site.
